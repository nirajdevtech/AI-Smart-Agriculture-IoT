# 🌱 AI-Powered Smart Agriculture Monitoring Platform

## 📌 Overview

AI-Powered Smart Agriculture Monitoring Platform is an IoT-based smart farming solution developed using Python, Flask, MQTT, HTML, CSS, and Chart.js.

The system simulates real-time agricultural sensors and provides automated irrigation decisions, crop health monitoring, analytics, and sensor history tracking through an interactive dashboard.

---

## 🚀 Features

- Real-Time Temperature Monitoring
- Real-Time Humidity Monitoring
- Soil Moisture Monitoring
- Automated Irrigation Control
- MQTT-Based Communication
- Crop Health Assessment
- Water Saving Analytics
- Sensor History Tracking
- Interactive Dashboard
- Smart Agriculture Analytics
- Real-Time Sensor Visualization

## ▶ How to Run

Install dependencies:

```bash
pip install flask paho-mqtt
```

Run Sensor Simulator:

```bash
python sensor_simulator.py
```

Run Dashboard:

```bash
python app.py
```

Open Browser:

```text
http://127.0.0.1:5000
```
